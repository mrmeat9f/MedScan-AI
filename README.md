# МедСканt — Интеллектуальная домашняя аптечка 💊✨

Современное мобильное Android-приложение для удобного ведения домашней аптечки, контроля сроков годности лекарств и планирования их приёма на базе искусственного интеллекта.

---

## 🎨 Ключевые возможности

### 1. Добавление лекарств на любой вкус ➕
При нажатии на кнопку добавления на главном экране пользователю предоставляется выбор из двух удобных вариантов:
*   **ИИ-сканер (По фото упаковки)** — сфотографируйте упаковку препарата, а встроенный ИИ автоматически распознает коммерческое название, действующее вещество, серийный номер и срок годности.
*   **Ручной ввод** — детальная и структурированная Material 3 форма с возможностью вручную указать название, дозировку, количество таблеток, график приема и установить дату окончания срока годности.

### 2. Продвинутое ИИ-распознавание (Gemini API) 📸🤖
*   Интегрировано современное семейство моделей **Gemini 3.5**.
*   **Оптимизация трафика и памяти**: Изображения с камеры автоматически сжимаются и масштабируются перед отправкой на сервер, что ускоряет распознавание и экономит интернет-трафик.
*   **Отказоустойчивость (Resilience)**: Внедрён механизм умных автопопыток при временных перегрузках API (ошибки `HTTP 429 - Too Many Requests` и `HTTP 503 - Service Unavailable`), гарантируя надёжную работу ИИ даже при высокой нагрузке на сервера.

### 3. Поиск аналогов лекарств 🔍
Если препарат закончился или его нет в аптеке, вы можете мгновенно запустить интеллектуальный поиск терапевтических замен и безопасных аналогов по активному веществу.

### 4. Контроль приёма и совместный доступ 📅🤝
*   Гибкий график регулярного приёма и приёма по необходимости (*as needed*).
*   Встроенный механизм синхронизации для совместного использования одной аптечки членами семьи.
*   Цветовая индикация состояния лекарств — фокус на истекающих сроках годности и минимальном остатке.

## 🛠️ Архитектура и технологии

Приложение построено в соответствии с передовыми стандартами разработки под Android:

*   **Язык программирования**: **Kotlin** (100% современного строго типизированного кода).
*   **Пользовательский интерфейс**: **Jetpack Compose** (декларативный UI).
*   **Архитектурный паттерн**: **MVVM (Model-View-ViewModel)** с использованием `StateFlow` для реактивного обновления экрана.
*   **Локальное хранилище**: **Room Database** для транзакционного, надёжного хранения объектов лекарств и синхронизационных логов.
*   **Сетевые запросы**: **Retrofit 2** в связке с **Moshi** для эффективной десериализации JSON.
*   **Асинхронность**: **Kotlin Coroutines** и **Flow** для плавного выполнения фоновых задач, сетевого ввода-вывода и работы с БД без подвисания UI.

## 🎨 Дизайн и эргономика

Интерфейс строго следует спецификациям **Material Design 3 (M3)**:
*   **Цветовая палитра**: Специально разработанная мягкая ментоловая цветовая гамма (`MintPrimary`, `MintTeal`), дарящая ощущение чистоты и спокойствия в медицинском приложении.
*   **Доступность (Accessibility)**: Интерактивные элементы ввода и управления спроектированы в соответствии с требованиями Google Accessibility — размер сенсорной области (touch target) составляет не менее **48dp x 48dp**, все иконки и изображения снабжены осмысленными `contentDescription`.
*   **Сетка и Спейсинг**: Строгий 8dp grid для идеального баланса пустого пространства, способствующего быстрой визуальной читаемости.
*   **Edge-to-Edge поддержка**: Полноэкранный режим без швов с грамотной обработкой `WindowInsets` и шторки уведомлений.
